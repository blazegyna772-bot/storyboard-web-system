## P1 本地后端车架阶段

目标：建立独立本地后端，并建立可插拔车架。

范围：

- `server/` FastAPI 后端
- `/api/projects/*`
- LLM proxy
- image provider proxy
- 项目根目录路径由后端配置文件管理
- 项目扫描、创建、删除、保存全部由后端文件系统处理
- 建立 adapter / rulepack / pipeline registry 的最小目录和接口

原则：

- 第一阶段只迁移，不改业务行为。
- API 路径保持不变，避免前端大面积重写。
- 所有后续能力都按可插拔节点、规则包或适配器设计。

## P4 分镜生产阶段
