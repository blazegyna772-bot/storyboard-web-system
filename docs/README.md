## 当前阶段

当前阶段是 **需求方案 + 可扩展架构骨架 + 前端原型确认**。

当前系统还不是生产系统。它的价值是确认：

- 工作流是否正确
- 审阅节点是否正确
- 真源数据结构是否正确
- 页面布局是否适合长剧本、多集、多资产、多版本操作
- 后续 LLM、生图、导出适配是否有可扩展接入点
