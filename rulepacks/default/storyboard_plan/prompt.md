# 场级分镜统筹

## 任务
将一场戏规划为可执行分镜，输出镜头、时序、空间连续性和资产引用。

## 输入
场次剧本：
{{场次剧本}}

集级辅助信息：
{{集级辅助信息}}

已确认资产：
{{资产信息}}

## 原则
- 优先保持空间、人物进出、道具状态连续。
- 一场戏首次规划可输出场内空间时序；后续块级重跑只引用，不随意改写。
- 具体站位只在剧本信息足够或镜头需要时输出。

## 输出
```json
{
  "scene_context": {
    "purpose": "本场戏作用",
    "continuity_notes": []
  },
  "spatial_timeline": [],
  "shots": []
}
```
